# Docker Portfolio

Proyectos prácticos para aprender Docker Desktop desde cero, orientados a desarrollo fullstack y buenas prácticas de producción.

## Proyectos

### [Proyecto 1 — App Dockerizada](./proyecto-1-docker)
App de tareas con React + FastAPI + PostgreSQL orquestados con Docker Compose.

**Conceptos:** contenedores, redes internas, volúmenes, variables de entorno, healthchecks.

---

### [Proyecto 2 — Optimización con Multi-Stage Builds](./proyecto-2-optimizacion)
La misma app reescrita con imágenes optimizadas. Reducción del 82% en el frontend.

**Conceptos:** multi-stage builds, Nginx Alpine, imágenes de producción.
